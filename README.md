# **mogl3d-editor**

<div>MOGL3D Editor is a WYSIWYG editor that allows 3D files to be uploaded on the web. </div>
In version 1.xx, only the 3D file upload feature is provided, and the supported 3D file formats are as follows:</p>

* fbx
* gltf
* obj
